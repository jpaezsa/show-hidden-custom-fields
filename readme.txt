Makes hidden custom fields visible when editing a post

== Description ==

Custom fields are hidden by prefixing them with an underscore. Use this plugin to make them visible again. Thanks to ViperBond007 for the orginal solution found at http://www.viper007bond.com/2009/08/16/wordpress-trick-how-to-show-hidden-custom-fields/

Read more about it here : http://themergency.com/show-hidden-custom-fields-wordpress-plugin/

== Installation ==

1. Upload the plugin folder 'show-hidden-custom-fields' to your `/wp-content/plugins/` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can now see, edit or delete custom fields
